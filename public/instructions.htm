Task: Scoring Image

In the next pages, you will be asked to score a series of _ images.

All the images in this task have been created by human subjects participating in a scientific experiment. To guarantee the correctness of scientific research, please try to be as much accurate as possible.

Score each image by moving a slider on a scale from 0 (lowest grade) to 10 (highest grade) on four dimensions:
  • Creativity
  • Interestingness as a face
  • Abstractness
  • Overall appealing or quality

How the images have been created

Each image has been created in less than 60 seconds using a special computer interface. This interface is designed to draw images quickly, but with some limitations.

You will see a random sample of the created images in the next page.
