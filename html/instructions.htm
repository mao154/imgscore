Image categorization

In the next pages you will be shown a series of 50-60 images.

The images have been generated with a computer software by human subjects participating in a scientific experiment. For the correctness of scientific research, please try to be as much accurate as possible.

You are asked to assign each image to one of the following four categories:
Human face:
An image resembling a human face. Some traits can be missing, or be slightly distorted, or the face might look overall more like a caricature, or a cartooned version of a face.
Non-human face:
An image having some, or all, the characteristic traits of a face, but so distorted, or out of the normal positions (for example with the mouth above the eyes), that it cannot be human. For example, fantasy, animal, or alien faces would rather fit here than in the category above.
Any other known object or shape:
An image resembling a known object or shape, but not a face. For example, a car, a kitchen tool, or a full body - containing a face - would belong to this category. The image does not need to be a close representation of the object, but can just remind it.
Abstract lines:
An image not belonging to any of the categories above goes here.

Tags

Additionally, you can also type some words (tags) describing the salient features of the image. Good tags help to distinguish groups of similar images within the same category.
If you have used a tag in one image, please be consistent and use the same tags for all images containig the same salient feature.
